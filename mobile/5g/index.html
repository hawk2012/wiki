    📄 5G

    (Fifth Generation Mobile Network)

    🔹 Общее описание

    5G (пятое поколение мобильной связи) — это новейшая технология беспроводной передачи данных, обеспечивающая сверхвысокую скорость, минимальные задержки и поддержку огромного количества подключенных устройств. Основная цель 5G — создать универсальную цифровую инфраструктуру для IoT, искусственного интеллекта, автономных систем и других современных приложений.

    • Стандарт: IMT-2020
    • Разработчик: ITU (Международный союз электросвязи), 3GPP
    • Первое внедрение: 2019 год (Южная Корея, США, Китай)
    • Скорость передачи данных:
    • До 20 Гбит/с (теоретически)
    • Реальная скорость: от 100 Мбит/с до 1–3 Гбит/с
    • Задержка (latency):
    • До 1 мс (ultra-low latency)
    • Поддержка устройств: до 1 млн устройств на км²

    🔹 Основные технологии 5G

    Технология Описание
    NR (New Radio) Новый стандарт радиоинтерфейса, разработанный 3GPP
    mmWave (миллиметровые волны) Диапазоны свыше 24 ГГц, обеспечивают высокую скорость, но малый радиус действия
    Sub-6 GHz Частоты ниже 6 ГГц (например, 3.5 ГГц), обеспечивают баланс между скоростью и покрытием
    Massive MIMO Использование сотен антенн для увеличения ёмкости сети
    Beamforming Формирование направленных сигналов для повышения качества связи
    Network Slicing Виртуальные сети под конкретные задачи (например, IoT, телемедицина)
    Edge Computing Обработка данных рядом с источником для снижения задержек

    🔹 Режимы работы 5G

    Режим Полное название Описание
    NSA (Non-Standalone) Неавтономный режим Использует существующую 4G инфраструктуру для управления
    SA (Standalone) Автономный режим Полностью независимая сеть 5G без опоры на 4G

    🔹 Архитектура сети 5G (на примере)

    Основные компоненты:

    Компонент Описание
    UE (User Equipment) Пользовательское оборудование (смартфон, модем, IoT-устройство)
    gNB (Next Generation Node B) Базовая станция 5G
    NG-RAN (Next Generation RAN) Радиодоступная сеть 5G
    AMF (Access and Mobility Management Function) Управление доступом и мобильностью
    SMF (Session Management Function) Управление сессиями
    UPF (User Plane Function) Передача пользовательских данных
    UDM (Unified Data Management) Хранение информации о пользователях
    AUSF (Authentication Server Function) Аутентификация пользователей
    NEF (Network Exposure Function) Экспорт сетевых возможностей внешним приложениям
    AF (Application Function) Внешние приложения, взаимодействующие с сетью

    🔹 Преимущества 5G

    • Сверхвысокая скорость передачи данных
    • Минимальные задержки (до 1 мс)
    • Поддержка массового числа подключённых устройств
    • Высокая надёжность и отказоустойчивость
    • Поддержка новых приложений: AR/VR, автономные автомобили, умные города
    • Улучшенная энергоэффективность (для IoT)

    🔹 Ограничения 5G

    • Требует значительных инвестиций в инфраструктуру
    • mmWave имеют ограниченный радиус действия и чувствительны к препятствиям
    • Высокая стоимость развертывания и лицензирования частот
    • Ограниченное покрытие в удалённых регионах
    • Совместимость с предыдущими поколениями требует сложных решений (NSA)

    🔹 Примеры использования

    Сфера применения Примеры
    IoT / M2M Умные датчики, автоматизированные системы управления
    Автономные автомобили Обмен данными между автомобилями (V2X), навигация в реальном времени
    Телемедицина Удалённая диагностика, хирургические операции через интернет
    AR/VR Расширенная и виртуальная реальность в реальном времени
    Умные города Уличное освещение, дорожные камеры, система парковок
    Промышленная автоматизация Роботы, датчики, системы контроля качества
    Образование и онлайн-медиа Интерактивные лекции, стриминг в 4K/8K

    🔹 Частотные диапазоны 5G

    Диапазон Частота Особенности
    Sub-1 ГГц (FR1-Low) < 1 ГГц Дальнее покрытие, проникающая способность
    Sub-6 ГГц (FR1) 1–6 ГГц Баланс между скоростью и покрытием
    mmWave (FR2) > 24 ГГц Очень высокая скорость, короткий радиус действия

    🔹 AT-команды для работы с 5G-модемами

    Для управления 5G-модемами (Quectel, Telit, Sierra Wireless и др.) часто используются следующие команды:

    AT              # Проверка связи
AT+CGMI         # Производитель
AT+CGMM         # Модель устройства
AT+CIMI         # IMSI номер SIM-карты
AT+CSQ          # Уровень сигнала
AT+COPS?        # Информация о сети
AT+CREG?        # Регистрация в сети
AT+CGATT?       # Прикрепление к пакетной сети
AT+CGDCONT      # Настройка PDP контекста
AT+QIACT        # Активация контекста (Quectel)
AT+QIDEACT      # Деактивация контекста
AT+QISTATUS     # Статус подключения

    🔹 Отключение старых сетей в пользу 5G

    С развитием 5G, многие страны начинают постепенно отключать устаревшие сети:

    Поколение Состояние
    2G Отключен или отключается во многих странах
    3G Отключается в США, Европе, Японии
    4G Функционирует, развивается, будет использоваться параллельно с 5G долгие годы
