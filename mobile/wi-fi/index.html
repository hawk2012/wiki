    📄 Wi-Fi

    (Wireless Fidelity)

    🔹 Общее описание

    Wi-Fi — это технология беспроводного подключения к локальной сети и интернету, основанная на стандартах IEEE 802.11. Она позволяет устройствам (смартфоны, ноутбуки, планшеты, IoT-устройства) подключаться к интернету без проводов, используя радиосигналы в диапазонах 2.4 ГГц и 5 ГГц.

    • Тип: Беспроводная сеть (WLAN)
    • Стандарты: IEEE 802.11a/b/g/n/ac/ax/be и др.
    • Частотные диапазоны: 2.4 ГГц, 5 ГГц, 6 ГГц (Wi-Fi 6E)
    • Протоколы: TCP/IP, WPA/WPA2/WPA3, WEP, EAP
    • Максимальная скорость: до 9.6 Гбит/с (Wi-Fi 7)

    🔹 Основные стандарты Wi-Fi

    Стандарт Год выпуска Диапазон Макс. скорость Особенности
    802.11b 1999 2.4 ГГц 11 Мбит/с Первый популярный Wi-Fi
    802.11a 1999 5 ГГц 54 Мбит/с Выше скорость, меньше помех
    802.11g 2003 2.4 ГГц 54 Мбит/с Совместимость с b
    802.11n 2009 2.4 / 5 ГГц До 600 Мбит/с MIMO, более устойчивый сигнал
    802.11ac 2014 5 ГГц До 3.5 Гбит/с MU-MIMO, широкие каналы
    802.11ax (Wi-Fi 6/6E) 2019–2021 2.4 / 5 / 6 ГГц До 9.6 Гбит/с OFDMA, TWT, BSS Coloring
    802.11be (Wi-Fi 7) 2023+ 2.4 / 5 / 6 ГГц До 46 Гбит/с Multi-Link Operation, 320 MHz каналы

    🔹 Частотные диапазоны

    Диапазон Преимущества Недостатки
    2.4 ГГц Больший радиус действия, проникающая способность Более загруженный, низкая скорость
    5 ГГц Меньше помех, выше скорость Меньшая зона покрытия
    6 ГГц (Wi-Fi 6E) Широкие каналы, высокая пропускная способность Требуется новое оборудование, ограниченное покрытие

    🔹 Архитектура Wi-Fi сети

    Основные компоненты:

    Компонент Описание
    Клиентское устройство (STA) Ноутбук, смартфон, планшет и т.д.
    Точка доступа (AP) Роутер или модем, обеспечивающий беспроводной сигнал
    SSID (Service Set Identifier) Имя беспроводной сети
    BSSID MAC-адрес точки доступа
    Канал Диапазон частот, по которому передаются данные
    Шлюз (gateway) IP-адрес маршрутизатора (например, 192.168.0.1)
    DHCP-сервер Автоматическое выделение IP-адресов клиентам

    🔹 Режимы работы Wi-Fi

    Режим Описание
    Infrastructure Mode Устройства подключаются к точке доступа (наиболее распространённый)
    Ad-Hoc Mode Устройства соединяются напрямую друг с другом
    Mesh Mode Создание распределённой сети из нескольких точек доступа
    Bridge Mode Соединение двух сетей через Wi-Fi
    Repeater Mode Усиление сигнала от основной точки доступа

    🔹 Безопасность Wi-Fi

    Версия безопасности Описание
    WEP Устаревший протокол, легко взламывается
    WPA (Wi-Fi Protected Access) Повышенная безопасность, замена WEP
    WPA2 AES-шифрование, стандарт де-факто
    WPA3 Последняя версия, защита от перебора пароля, SAE-протокол
    Enterprise Аутентификация через RADIUS-сервер (используется в корпоративных сетях)

    🔹 Настройки Wi-Fi на роутере

    Пример базовых настроек:

    • SSID: имя вашей сети
    • Пароль (PSK): не менее 8 символов
    • Режим шифрования: WPA2-PSK или WPA3
    • Канал: авто или выбран вручную для минимизации помех
    • IP-адрес роутера: обычно 192.168.0.1 или 192.168.1.1
    • DHCP: включен по умолчанию
    • MAC-фильтрация: ограничение подключения по MAC-адресам

    🔹 AT-команды для работы с Wi-Fi модулями

    Для управления Wi-Fi модулями (например, ESP8266, ESP32, SIM7600, A6, A7 и др.) часто используются следующие команды:

    AT              # Проверка связи
AT+CWMODE=?     # Показать поддерживаемые режимы (STA/AP/STA+AP)
AT+CWMODE=1     # Установить режим STA (клиент)
AT+CWJAP="SSID","PASSWORD"   # Подключение к точке доступа
AT+CIFSR        # Получить IP-адрес
AT+CIPSTART="TCP","example.com",80  # Установить TCP-соединение
AT+CIPSEND      # Отправить данные
AT+CIPCLOSE     # Закрыть соединение

    🔹 Wi-Fi и технологии связи

    Технология Интеграция с Wi-Fi
    LTE / 5G Возможен переход между Wi-Fi и мобильным интернетом
    VoWiFi Голосовые вызовы через Wi-Fi (Wi-Fi Calling)
    IoT Поддерживается большинством умных устройств
    Wi-Fi 6/6E/7 Улучшенная производительность, энергоэффективность, низкие задержки

    🔹 Примеры использования

    Сфера применения Примеры
    Домашние сети Интернет, стриминг, онлайн-игры
    Офисные сети Подключение рабочих станций, принтеров
    Публичные точки Кафе, аэропорты, гостиницы
    Умный дом Умные колонки, камеры, термостаты
    Индустриальные системы Видеонаблюдение, автоматизация
    Образование Классы с цифровыми учебными материалами

    🔹 Распространённые проблемы и решения

    Проблема Возможное решение
    Слабый сигнал Переместите роутер ближе, используйте повторитель
    Высокая нагрузка Переключитесь на 5 ГГц, обновите роутер
    Не подключается Проверьте пароль, перезагрузите устройство
    Высокий пинг Используйте Ethernet, проверьте фоновые приложения
    Не видит сети Включите Wi-Fi, проверьте драйверы
    Плохая скорость Измените канал, обновите прошивку роутера
