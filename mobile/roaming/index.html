    📄 Роуминг

    (Roaming)

    🔹 Общее описание

    Роуминг (roaming) — это возможность использования мобильной связи за пределами домашней сети, когда абонент подключается к сети другого оператора в другой стране или регионе. Это позволяет сохранять связь и доступ к услугам (звонки, SMS, интернет) вне зависимости от географического положения.

    • Тип: Услуга мобильной связи
    • Поддержка: GSM, UMTS, LTE, 5G
    • Стандарты: 3GPP TS 23.060 (GPRS), TS 23.272 (LTE), TS 29.501 (5G)
    • Протоколы: MAP, GTP, Diameter, SIP

    🔹 Основные виды роуминга

    Вид роуминга Описание
    Национальный Использование услуг другого оператора внутри страны
    Международный Подключение к сети иностранного оператора
    Гостевой (visited) Использование услуг сети при нахождении в роуминге
    Выходящий (outbound) Ваш оператор предоставляет услуги своим клиентам в других сетях
    Входящий (inbound) Ваша сеть обслуживает клиентов других операторов

    🔹 Как работает роуминг?

    Основные этапы:

    1. Регистрация устройства:
    2. UE (устройство) сканирует доступные сети.
    3. Выбирается сеть по настройкам (ручной/автоматический выбор).

    4. Отправляется запрос регистрации (Location Update Request).

    5. Проверка доступа:

    6. VLR (Visited Location Register) запрашивает данные у HLR (Home Location Register).

    7. AUC (Authentication Center) проверяет IMSI и ключ авторизации.

    8. Установление соединения:

    9. Для голосовых вызовов: через MSC/VLR.

    10. Для данных: через SGSN/GGSN (в 3G), SGW/PGW (в LTE), UPF (в 5G).

    11. Передача данных о тарификации:

    12. CDR (Call Detail Records) передаются домашнему оператору для биллинга.

    🔹 Архитектура роуминга (на примере GSM / LTE)

    GSM:

    MS → BTS → BSC → MSC/VLR → HLR/AUC → Домашний оператор

    LTE:

    UE → eNodeB → MME → HSS → PGW → Домашний оператор

    Основные компоненты:

    Компонент Описание
    HLR/HSS Хранение информации о пользователе
    VLR Временное хранение данных о роумирующих пользователях
    MSC Центр коммутации вызовов
    SGSN / MME Управление данными и мобильностью
    GGSN / PGW / UPF Шлюз к внешним сетям (интернет)
    Billing System Система расчёта стоимости услуг роуминга

    🔹 Типы услуг в роуминге

    Услуга Поддерживается в роуминге Примечание
    Голосовые вызовы ✅ Могут быть ограничения
    SMS ✅ Может использоваться SMS over IP
    MMS ✅ Зависит от настроек APN
    Интернет (GPRS/LTE) ✅ Требуется корректная настройка APN
    VoLTE ⚠️ Иногда Требует поддержки IMS и договорённости между операторами
    USSD-сервисы ❌ Редко Часто недоступны в роуминге

    🔹 Настройки APN для интернета в роуминге

    Примеры стандартных APN-настроек:

    Оператор APN Логин Пароль Примечание
    МТС (Россия) internet.mts.ru mts mts Автоматически
    Билайн (Россия) internet.beeline.ru beeline beeline
    Мегафон (Россия) internet.gprs.mgf gprs gprs
    Vodafone (ЕС) internet.vodafone.net — —
    Turkcell (Турция) internet — —

    Если интернет не работает, проверьте настройки APN в разделе: Настройки → Мобильная сеть → Точки доступа (APN)

    🔹 AT-команды для работы с роумингом

    AT              # Проверка связи
AT+CGMI         # Производитель модема
AT+CGMM         # Модель устройства
AT+CIMI         # IMSI номер SIM-карты
AT+CSQ          # Уровень сигнала
AT+COPS?        # Информация о текущей сети
AT+CREG?        # Статус регистрации в сети
AT+CGATT?       # Прикрепление к пакетной сети
AT+CUSD=1,"*100#"  # Отправка USSD-запроса

    🔹 Преимущества роуминга

    Преимущество Описание
    Мобильность Возможность использовать телефонную связь и интернет в любой точке мира
    Безопасность Не нужно менять SIM-карту, остаётся ваш номер
    Интеграция с биллингом Все расходы учитываются в одном счёте
    Поддержка IoT Устройства могут работать в разных странах без замены SIM

    🔹 Недостатки и ограничения

    Ограничение Описание
    Высокая стоимость Роуминг может быть дороже локального тарифа
    Ограничения на использование Некоторые операторы блокируют определённые сервисы
    Зависимость от соглашений между операторами Нет гарантии подключения к конкретной сети
    Проблемы с покрытием Не всегда доступны LTE/5G в удалённых регионах
    Отсутствие USSD и дополнительных сервисов Часто не работают в роуминге

    🔹 Роуминг и технологии связи

    Технология Поддержка роуминга
    2G Да (SMS, звонки, GPRS)
    3G Да (HSPA+, MMS, интернет)
    4G LTE Да (VoLTE, высокая скорость)
    5G Да (NSA и SA режимы)
    IoT / NB-IoT / LTE-M Да (при наличии поддержки оператора)

    🔹 Роуминг в Европейском Союзе (EU Roaming)

    • "Roam Like at Home" — действующая политика ЕС:
    • Абоненты платят те же деньги, что и дома
    • Без дополнительных сборов за роуминг
    • Ограничения:
      • Разумное использование (fair use policy)
      • Некоторые операторы применяют лимиты

    🔹 Экономия в роуминге

    Способ Описание
    eSIM Использование второй SIM с локальным тарифом
    Локальные SIM-карты Покупка SIM-карты на месте
    Wi-Fi + VoIP Использование Wi-Fi и мессенджеров (Skype, WhatsApp)
    Мобильные держатели SIM Устройства с несколькими SIM-картами
    Тарифы с низким роумингом Например, Tele2, YOTA, T-Mobile One
