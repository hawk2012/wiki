📄 SelectOS

🧾 Общее описание

SelectOS — это легковесная, стабильная и безопасная операционная система, основанная на Debian. Система предназначена для использования в облачных и серверных инфраструктурах, где важны надёжность, минимализм и производительность. Установка и использование происходят исключительно через командную строку (CLI), графическая оболочка (GUI) не предусмотрена.

Разработана специалистами Selectel для оптимизации работы в их облачной платформе и других высоконагруженных сценариях.

🖥 Технические требования

Минимальные системные требования:

Компонент Требование
Архитектура процессора x86_64 (64-bit)
Частота процессора Не менее 2 ГГц
Оперативная память Не менее 512 МБ
Место на диске Не менее 5 ГБ

⚠️ Рекомендуется использовать более мощное оборудование для продакшн-сред.

🔒 Особенности системы

  • Базовая установка без GUI
  • Ядро Linux (LTS версии)
  • Полностью совместима с пакетами .deb
  • Поддерживает автоматическое обновление безопасности (APT)
  • Интеграция с инструментами мониторинга и управления Selectel
  • Настроенная минимальная поверхность атак: только необходимые сервисы
  • Простая и быстрая настройка через CLI

💾 Установка

SelectOS может быть установлен:

  • В виде образа ISO (для физических или виртуальных машин)
  • Через облачный API Selectel (Cloud Platform)
  • Как custom image в других гипервизорах

Шаги установки (кратко):

  1. Загрузите образ с официального сайта или через Cloud Control Panel.
  2. Загрузите его как виртуальный диск или запишите на USB/CD.
  3. Выполните загрузку с носителя.
  4. Следуйте указаниям текстового установщика.
  5. Настройте сеть, имя хоста, пользователя и SSH-доступ.

🛠 Системное ПО

  • Пакетный менеджер: apt / apt-get
  • Инициализация: systemd
  • Язык интерфейса: русский (по умолчанию)
  • Поддерживаемые файловые системы: ext4, xfs, btrfs (опционально)

🔐 Безопасность

  • SELinux/AppArmor (настраивается)
  • Регулярные обновления безопасности
  • Отключён root-логин по SSH по умолчанию
  • Настроенные правила iptables/nftables

🌐 Сетевые возможности

  • DHCP и статическая настройка сети
  • DNS-клиент (resolvconf / systemd-resolved)
  • Поддержка VLAN, bonding, bridging
  • Интеграция с VPC (в облаке Selectel)

📦 Примеры установки программ

sudo apt update
sudo apt install nginx

sudo apt install python3 git curl

📁 Файловая система

  • / (root) – основная точка монтирования
  • /home – пользовательские данные
  • /var/log – логи
  • /etc – конфигурационные файлы

📞 Поддержка

  • https://selectel.ru — официальный сайт
  • https://selectel.ru/services/cloud/private-cloud/ — облачная платформа
  • support@selectel.ru — техническая поддержка

📝 Лицензирование

SelectOS предоставляется бесплатно. Использование и распространение осуществляется в рамках лицензий используемых компонентов (GPL, LGPL, Apache и т.д.).

✅ Версии и релизы

  • Стабильная ветка: соответствует LTS-релизам Debian
  • Обновления: через apt upgrade
  • Частота релизов: 1 раз в 6 месяцев (минорные обновления), major — каждые 2 года
