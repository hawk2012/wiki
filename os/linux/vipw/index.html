vipw

Описание

Команда vipw (от англ. edit password file) используется в Unix-подобных операционных системах для безопасного редактирования файлов /etc/passwd и /etc/shadow. Она позволяет вручную изменять информацию об учётных записях пользователей с блокировкой файлов, чтобы избежать конфликтов при одновременном редактировании.

Синтаксис

vipw [опции]

Опции

Опция Описание
-g, --group Редактирует файл /etc/group вместо /etc/passwd
-s, --shadow Также открывает файл /etc/shadow для редактирования
-h, --help Показывает справку по использованию команды

Примеры использования

  1. Открыть файл /etc/passwd: bash sudo vipw

  2. Открыть файл /etc/shadow: bash sudo vipw -s

  3. Открыть файл /etc/group: bash sudo vipw -g

Важно

  • Команда требует прав суперпользователя (root), так как редактируемые файлы защищены.
  • Используется в основном опытными системными администраторами.
  • Некорректное редактирование может повлечь нарушение безопасности или работоспособности системы.

Предостережения

  • Не рекомендуется редактировать эти файлы напрямую.
  • Перед редактированием всегда делайте резервные копии.
